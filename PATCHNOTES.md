# NAI Auto Generator V4.5 패치 노트

---

## V4.5_2.6.04.28 (2026-04-28)

### 🔁 세팅별 연속 생성 버튼 UI 노출

- 기존에 내부적으로만 존재하던 **"세팅별 연속 생성"** 버튼을 이미지 버튼 툴바에 정식 노출
- 여러 설정 파일(`.txt`)을 선택하면 각 설정을 순차적으로 적용하며 배치 생성 수행
- 생성된 이미지는 **설정 파일명 기반 서브폴더**에 자동 저장
- 연속 생성 대화상자(생성 횟수·간격·오류 처리)를 그대로 활용
- 활용 예시: 캐릭터별 일괄 생성, 해상도 비교 생성 등

## V4.5_2.6.04.23 (2026-04-23)

### 🔍 Danbooru 태그 자동완성 개선

- 태그 목록을 **인기도(post_count) 내림차순**으로 정렬하여 자주 쓰이는 태그가 상위에 표시
- **공백 ↔ 언더스코어 호환 검색** 지원: `blue sky` 입력 시 `blue_sky` 태그도 검색됨
- 검색 방식 분리: **prefix 매칭**(최대 50개) + **contains 매칭**(최대 50개)으로 관련 결과를 더 넓게 제공
- 내부 구조 개선: `TagData` dataclass 도입으로 태그 데이터 관리 일원화

## V4.5_2.6.04.14 (2026-04-14)

### 🖥️ UI 대규모 개선

#### Options & Settings 통합 패널

- "Image Options"와 "Advanced Settings" 두 개의 독립 패널을 하나의 **"Options & Settings"** 그룹박스로 통합
- QGridLayout 기반의 4컬럼 효율적 배치로 좁은 화면에서도 정상 표시:
  - 1행: 모델 선택 (전체 너비)
  - 2행: 해상도 프리셋 선택
  - 3행: 너비(W) / 높이(H) / Random 체크박스
  - 4행: 샘플러 · 노이즈 스케줄
  - 5행: Steps · Prompt Guidance (CFG Scale)
  - 6행: Seed/Fix/Random · Prompt Guidance Rescale
  - 7행: Variety+ · Legacy Mode
- 기존의 수직 구분선 제거

#### 결과 프롬프트 오버레이

- 결과 프롬프트 표시 방식 변경: 별도 하단 패널 → **이미지 위 반투명 오버레이**
- 결과 이미지 하단 바에 **"프롬프트 결과 표시"** 체크박스 추가로 표시/숨김 토글
- `Files` → `Option` → 프롬프트 설정에서 **오버레이 텍스트 색상** 커스터마이징 가능

#### Generate 버튼 위치 변경

- "1회 생성", "연속 생성" 버튼이 좌측 패널에서 **결과 이미지 영역 하단 바**로 이동
- 하단 바 구성 (왼쪽 → 오른쪽):  
  `[1회 생성]` `[연속 생성]` `|` (여백) `[이미지 저장]` `[크기 초기화]` `[☐ 프롬프트 결과 표시]`
- "세팅별 연속 생성" 버튼은 하위 호환성 유지를 위해 내부적으로 존재하나 UI에서 숨김

### 🤖 지원 모델 확장

기존 3개에서 **5개**로 확장:

| 모델 ID | 표시 이름 | 비고 |
|---------|----------|------|
| `nai-diffusion-4-5-full` | NAI Diffusion 4.5 Full | 기본값 |
| `nai-diffusion-4-5-curated` | NAI Diffusion 4.5 Curated | |
| `nai-diffusion-4-full` | NAI Diffusion 4 Full | |

- V3 모델 선택 시 V4 전용 파라미터(`v4_prompt`, `characterPrompts`, `v4_model_preset` 등) 자동 제외하여 API 호환성 보장


### 🔑 API 키 로그인 지원

- 로그인 대화상자에 **"API 키"** 탭 추가 (기존 ID/Password 탭과 병행)
- NovelAI Persistent API Token (`pst-...`)으로 직접 로그인 가능
- 토큰 취득 방법: NovelAI 설정 → Account → "Get Persistent API Token"
- 토큰 표시/숨김 토글 버튼 제공

### 📝 프롬프트 UI 개선

- Prompt / Negative Prompt 입력 영역을 **탭(Tab) UI**로 전환
- 각 탭 제목에 현재 문자 수 실시간 표시 (예: `Prompt (248)`)
- 긴 프롬프트 작성 시 가독성 향상

### 👥 캐릭터 프롬프트 레이아웃 변경

- 캐릭터 프롬프트 패널을 수평 배열에서 **수직 스크롤** 레이아웃으로 변경
- 6개 캐릭터 추가 시에도 UI 오버플로우 없이 스크롤로 접근 가능

### 🌐 네트워크 상태 팝업 제거

- 네트워크 연결/해제 시 팝업 알림 제거
- 상태바에 조용하게 업데이트되도록 변경 (방해 최소화)

### 🔧 코드 품질 개선

- JSON 언어 파일 중복 키 제거
- mutable 기본 인자 버그 수정
- 4개 언어 파일(ko/en/ja/zh) 전체 i18n 누락 항목 보완
- bare except 정리 → 명시적 예외 처리로 전환
- danbooru_tagger 입력 검증 로직 추가
- 로깅 통일: `logging.xxx()` → `logger.xxx()`
- 중복 캐릭터 프롬프트 재구성 코드를 공유 메서드로 추출
